# template-cli

模板-脚手架

## 配置

1. 克隆项目

```bash
git clone https://github.com/rainbow-design/template-cli.git
```

2. 安装依赖

```bash
npm install
```

3. 链接全局命令

```bash
# 此命令可能需要管理员权限
npm link
```

## 使用

rainbow -h

```
Options:
  -V,-v,--version  output the version number
  -h, --help       display help for command

Commands:
  init             初始化一个项目
  update           更新脚手架
  tmp              查看可选模板
```
